# mfuleni-motor-spares.github.io
Official website for Mfuleni Motor Spares, providing automotive parts and accessories to mechanics, panel beaters, and vehicle owners in Mfuleni, Cape Town.

**Mfuleni Motor Spares Website**

This is an editable website for Mfuleni Motor Spares.

**How to edit**

Open the folder in Visual Studio Code.
Main files:
`index.html` = home page and main sections
`products.html` = product catalogue
`gallery.html` = gallery page
`contact.html` = contact and map page
`css/style.css` = colours, layout, design
`js/script.js` = chatbot and product search
`products.json` = easiest place to add, remove, or edit products
`images/` = logo and photos

**How to publish for free using GitHub Pages**

Create a GitHub account.
Create a new repository called `mfuleni-motor-spares`.
Upload all files in this folder.
Go to Settings > Pages.
Under Source, choose `Deploy from a branch`.
Choose branch `main` and folder `/root`.
Save.
GitHub will give you a free website link.

**How to collect reviews and messages**

The forms are ready for Formspree.
Go to https://formspree.io
Create a free form.
Copy your form endpoint.
Replace `https://formspree.io/f/YOUR-FORM-ID` in `index.html`.
